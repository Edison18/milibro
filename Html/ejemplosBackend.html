Ejemplos sencillo de uso de una tecnología backend





Para este ejemplo he escogido el ejemplo comentado en el foro número 1 antes presentado, donde procedí a explicar de manera breve y sencilla como se podría realizar un ejemplo del compún hola mundo.

Para este caso utilizamos la aplicación XAMPP (click aquí para ir a página oficial), misma que es una versión multiplataforma que contiene la gestión de bases de datos MySQL, la base de datos MariaDB, soporta lenguajes de prgramación como PHP y Perl y como servidor web usa Apache.

Vídeo explicativo del uso de XAMPP

De forma breve en el vídeo se empieza por comentar acerca de XAMPP, así como los mayores conocidos que usan Apache como servidor web Apache, procedemos a realizar un mínimo código en el IDE de Visual Studio Code en el lenguaje de prgramacipon de PHP, guardamos este documento en la carpeta "htdocs" que se encuentra dentro de la caprte de instalación de XAMPP a la que podemos acceder generalmente yendo a windows C / XAMPP / htdocs ... y en esta podemos guardar nuestro archivo con el nombre de index.php. Así solo ingresamos a XAMPP y levantamos los servicios y procedemos a ingresar al documento, mismo que nos cargará si todo está correcto.
